# 概要 #
donut(v1.6) の com.android.launcher の移植版。<br>
そのまま、コピーして、ユーザアプリとして、コンパイルを通るようにしたもの。<br>
ユーザアプリからアクセスできないシステム・プログラム・ファイルは、<br>
launcher と同じようにコピーしてコンパイルを通るようにした。<br>
